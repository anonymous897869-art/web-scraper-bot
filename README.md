# 🌐 Web Scraper Bot

A Python automation tool that extracts and processes data from websites using web scraping techniques.

---

## 🚀 Features
- Extracts data from web pages
- Uses requests to fetch HTML content
- Parses HTML using BeautifulSoup
- Displays extracted quotes in a structured format
- Beginner-friendly automation project

---

## 🛠️ Tech Stack
- Python 3
- Requests
- BeautifulSoup4

---

## ⚙️ How It Works
1. Sends HTTP request to target website  
2. Receives HTML response  
3. Parses HTML using BeautifulSoup  
4. Extracts required elements  
5. Displays output in console  

---

## ▶️ How to Run
```bash
python src/main.py
## 📦 Installation

```bash
pip install -r requirements.txt
